## 注意事项

- 测试环境优先**按量付费**，用完即停；集中测试 1–2 个月则包月 ¥89 更划算。
- 正式环境建议直接选 **1年或3年**，3年付月均仅 ¥60，比逐月续费节省约 30%–50%。
- 用户量到 5000–8000 以上时可考虑升配或水平扩展，阿里云支持按剩余时长补差价升配。
- 新用户秒杀价续费通常涨价 3–5 倍，长期使用优先选"新老同享+续费同价"机型。
- 国内节点建站需备案（约 20 个工作日）；香港节点无需备案。
- 促销价格随时可能调整，购买前请至官网核实最新价格。
