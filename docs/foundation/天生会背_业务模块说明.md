# 天生会背 · 系统业务模块说明

> 本文从业务角度介绍系统模块拆分和系统功能组成。

## 系统总览

-> 模块 -> 每个具体的功能

天生会背是一套面向英语单词学习的在线平台，服务三类用户：

- **学生**：每天打卡、做练习、积累词汇
- **老师**：建词表、布置任务、跟踪学生学情
- **平台管理方**：维护词库题库、管理用户、配置规则

整个系统划分为 **12 个业务模块**：

| 模块               | 一句话说明                          | 主要使用者                  |
| ------------------ | ----------------------------------- | --------------------------- |
| 1. 账号与身份      | 三种身份注册登录与个人信息          | 学生 / 老师 / 管理方        |
| 2. 标准智能词库    | 平台统一维护的权威词库与词表        | 管理方维护，全员使用        |
| 3. 非标准智能词库  | 用户自建词库，经审批后可用          | 学生 / 老师创建，管理方审批 |
| 4. 多维例句        | 例句与词库词条的多维关联与分级译文  | 管理方维护，全员使用        |
| 5. 词表与学习集    | 自由组织和管理学习内容              | 学生 / 老师                 |
| 6. 学习与练习      | 按题型智能出题并完成练习 + 每日打卡 | 学生 / 老师                 |
| 7. 教学任务        | 老师 / 学生创建与布置学习任务       | 老师 / 学生                 |
| 8. 班级            | 班级创建、成员管理与邀请加入        | 老师 / 学生                 |
| 9. 学情数据中心    | 看清学习效果与薄弱环节              | 老师 / 管理方               |
| 10. 天生币积分体系 | 激励活跃、内容贡献变现              | 全员                        |
| 11. 邀新推广与互动 | 老带新增长 + 社区互动               | 全员                        |
| 12. 平台运营后台   | 管理方的总控制台                    | 管理方                      |

---

## 模块详解

### 模块 1 · 账号与身份

- 手机号验证码 / 邮箱两种方式注册登录
- 学生、老师、管理方三种身份，进入各自的功能界面
- 个人信息维护：头像、昵称、所在学校
- 协议与隐私、退出登录
- 成为老师，（身份证，学历，接阿里云服务）
- 绑定邮箱，绑定手机号，补充个人信息

**周期：** : 14天

---

### 模块 2 · 标准智能词库

平台方在后台创建与维护的权威词库，侧重审核与验收，是全平台学习内容的基准数据源。

**后台创建 · 基础信息**

- 释义（多维释义）、词性、音标
- 参考资料、校正音频、教材版本（国内教学场景必需）
- 真人 / 智能发音，支持英式 / 美式
- 词汇结构细化：按"语境"和"词义分组"组织释义，让出题更精准
- 连读配置：可开关连读符号，前端按配置显示或隐藏

**难度等级管理**

- 按年级：一级（小学三年级～初中一年级）、二级（初中二年级～高中一年级）、三级（高中二年级～高考）、四级（进阶）
- 按 CEFR：A1、A2、A3、B1、B2、C1、C2 共 6 级

**COCA 语料库**

- 雅思 / 托福词汇、专业英语（化学、物理、经济等）
- 词频数据支撑分级与选题

**学习集调用**

- 对接第三方（科大讯飞），自动生成单词发音

**标准智能词表**

- 搜索栏：按条件检索词表
- 词表列表：查看词表详情
- 操作：查看、删除

**周期：** 14 天

---

### 模块 3 · 非标准智能词库

用户自建词库，与身份绑定，支持自定义词条的增删改查与发布，须经管理方审批后纳入可用范围。

**用户词库**

- **学生词库**：学生个人创建的自定义词库
- **老师词库**：老师个人创建的自定义词库

**操作**

- 查看、删除、审批（管理方）

**周期：** 6 天

---

### 模块 4 · 多维例句

将例句与智能词库中的单词、短语、词义进行多维关联，支撑分级学习与精准出题。与标准 / 非标准智能词库配合使用。

**例句录入**

- 录入英文例句，支持例句音频播放
- 例句中含英式 / 美式拼写差异时，分别提供 BrE、AmE 两套发音按钮
- 按 CEFR 等级（A1～C2）标注例句适用难度；不同等级学生看到不同复杂度的例句

**成分用词 · 关联匹配**

- 在例句中标记成分用词：关联单词、关联短语、清除格式
- 点击「开始匹配」：系统在智能词库中模糊检索，匹配成功则下划线标注，未匹配则提示「未找到相关词汇」
- 维护人员为每个成分选择对应词义（默认取该词条下第一条最简释义）
- 配置方言及词形：BrE / AmE 分别绑定具体词形（如过去式 centred / centered）
- 短语可继续拆解为子成分，逐词关联词义与词形

**译文管理**

- 支持汉语译文，按初阶 / 中阶 / 高阶分级录入多条译文
- 默认展示中阶译文，学习端可切换难度层级

**联动与出题**

- 保存后，例句自动归入所有已关联词条的例句列表
- 出题时按当前考查的词义选取对应例句，并自动加粗该词义下的正确词形
- 学习端按学生偏好自动切换英式 / 美式词形

**周期：** 10 天

---

### 模块 5 · 词表与学习集

- 老师 / 学生自己创建词表
- 词表可设公开或私密；喜欢的公开词表可一键收藏
- 班级词表、老师推荐词表，学生直接取用
- "学习集"：把自建的、收藏的、班级和老师推荐的内容集中管理
- 支持按字母、词频、自定义排序；简约 / 完整两种查看模式

**周期：** 14 天

---

### 模块 6 · 学习与练习

选定词表后，系统按题型自动出题并完成练习，老师无需手工编题。

**智能出题（通用规则）**

- 根据单词是否依赖上下文，自动选择合适的出题方式
- 干扰选项智能生成：用同类词混淆，排除意思过于接近的选项
- 单词与短语通过语法结构关联后，可进一步生成语法结构类题目

**单词类题型**

- **跟读**：听发音、跟着读，系统自动给口语打分
- **单选**：听音 / 看词选释义
- **听写**：听发音写单词，写错的字母会标出
- **翻译**：中英互译，支持手写
- **拼写**：看中文释义，拼写对应英文

**语法结构类题型**

单词与短语加入语法结构后，按以下题型出题：

- **跟读题**：播放语法结构中加粗部分，固定文字作分隔
- **单选题**：与单词类相同，按是否依赖上下文决定出题逻辑
- **听写题**：播放语法结构音频并展示中文释义，用户填写蓝色英文部分；写错的字母标出（黑色书写也判错，整题记错）；多个语法结构释义相同时合并练习
- **多选题**：仅当单个语法结构对应多个中文释义时触发；正确选项为该结构全部释义，干扰项从同词表、同词性子类中随机抽取，排除近义词
- **拼写题**：展示中文释义，拼写蓝色英文部分；拼写错误标出；练习结束后展示正确语法结构

**练习与打卡**

- **每日打卡**：完成老师布置的当日任务
- 单题作答或批量作答，灵活选择
- 练习记录可随时回看

**周期：** 20 天

---

### 模块 7 · 教学任务

老师布置或学生自建学习任务，选定词表与题型后进入练习，并查看任务数据统计。

**老师 · 长期任务**

- 创建任务 → 完善任务名称 → 选择任务类型「长期任务」
- 选择任务对象：整个班级或指定学生
- 选择词表、选择题型 → 完成任务创建
- 进入任务 → 进行练习 → 查看数据统计
- 任务内容可随时更新——后台调整词汇后，学生收到的题目同步更新
- 任务开始后，学习进度与正确率统计重新计算

**老师 · 每日任务**

- 创建任务 → 完善任务名称 → 选择任务类型「每日任务」
- 选择任务对象：班级或学生
- 选择词表、选择题型 → 配置每日题量 → 配置任务截止时间
- 系统每天凌晨 4:00 自动刷新当日进度
- 进入任务 → 进行练习 → 查看数据统计

**学生 · 自定义长期任务**

- 创建任务 → 完善任务名称 → 选择「长期任务」
- 选择词表、选择题型 → 完成任务创建
- 进入任务 → 进行练习 → 查看数据统计

**学生 · 每日任务**

- 创建任务 → 完善任务名称 → 选择「每日任务」
- 选择词表、选择题型 → 配置每日题量 → 配置任务截止时间
- 系统每天凌晨 4:00 自动刷新当日进度
- 进入任务 → 进行练习 → 查看数据统计

**周期：** 14 天

---

### 模块 8 · 班级

班级的创建、成员管理与邀请加入；班级是任务发放的对象载体，与教学任务模块配合使用。

**老师端**

- 搜索班级：按班级名称 / 班级码 / 校区检索
- 创建班级，完善班级信息（班级名称、班级码、所属校区）
- 班级卡片展示：词表数量、成员数量、关联任务数量
- 处理学生加入申请（如有待审批，卡片上提示申请人数）
- 分享链接或二维码，邀请学生加入班级
- 班级关联通用方法等级列表后，系统自动生成长期班级任务

**学生端**

- 搜索：按课程名称 / 班级 / 老师名称检索
- 展示当前已加入的全部班级（班级名称、班级码、创建老师、课程数、学生数）
- 通过老师邀请进入班级：展示老师信息与目标班级，学生可选择「确定」加入或「拒绝」
- 也可主动点击「加入班级」通过班级码等方式加入

**周期：** 10 天

---

### 模块 9 · 学情数据中心

- 学习进度、答题正确率
- **学情预警**：自动提示"哪些学生、哪些单词错误率偏高"
- **薄弱点统计**：本周全班错得最多的 Top10 单词
- 任务完成率统计
- 平台方可按时间段查看整体人员数据

---

### 模块 10 · 天生币积分体系

- **获得积分**：每日首次登录、完成打卡任务（按完成题量给分）、自己的词表被别人点赞、成功邀请新用户
- **使用积分**：创建带音频的自定义词表会消耗积分；也可以把积分"投"给其他用户
- 余额、明细随时可查
- 优质内容贡献者（如词库维护人员）的积分可与平台兑换，形成贡献回报

> 说明：因为涉及"积分兑换"等价值流转，上线前建议就奖励规则、防刷机制、兑换方式做一次专门的规则确认。

---

### 模块 11 · 邀新推广与互动

- **邀新推广**：用户生成专属邀请码 / 邀请海报，新用户凭码注册，推广人获得积分奖励
- 邀新记录可查，清楚看到自己邀请了谁
- **互动**：用户可对词表点赞、发表评论(参考b站结构）。

---

### 模块 12 · 平台运营后台

- **标准词库管理**：维护标准智能词库、设置难度与分类、管理标准词表
- **非标准词库管理**：审批用户词库、查看 / 删除学生与老师词库
- **多维例句管理**：维护例句、成分关联、分级译文
- **题库管理**：管理题型、题目分类、组题
- **用户管理**：管理老师与学生，支持批量导入、启用 / 停用
- **积分规则配置**：设定积分如何获得、如何消耗
- **任务推送配置**：控制任务内容的更新与推送
- **评论管理**：查看与清理评论
- **系统设置**：分角色分权限、系统参数配置
- **专职词库维护角色**：可设立专门维护词库的管理账号，与平台主账号分工
- 联系客服，新用户引导。
